WebSPC/webspc-frontend

项目介绍
WebSPC是SPC(统计过程控制)的web实现，采用前后端分离的方式。开源且免费。

项目架构
前端：
    框架：Vue3 + Vite + Pinia
    UI: Element-Plus
    绘图库：Plotly.js
    语言: TypeScript
